# vue-jwt
## Introduction
- vue-jwt is a front-end solution for PhoneBook-backend. It is based on Vue.js with Vuex and Vue Router
- Combined with JWT Authentication in Spring Boot

### Technology
- vue
- vue-router
- vuex
- axios
- vee-validate
- bootstrap
- vue-frontawesome
